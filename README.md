# LoginAP – Employee & Task Management App

⚠️ Work in Progress

## Description
LoginAP is a desktop application for managing employees and tasks.
The application includes role-based access, a graphical user interface,
and an SQL database backend.

Some parts of the application are fully functional, while others are
still under development.

## Implemented Features
- Login system with user roles (Admin, Employee, Intern)
- Role-based navigation and UI
- Employee management (Add / Update / Delete)
- Task management (partially implemented)
- SQL database connection (PostgreSQL)
- Modular architecture (Controller / View / Model)

## Work in Progress
- Some modules are UI-only (logic not completed yet)
- Additional database features planned
- Code refactoring and cleanup

## Technologies
- Python
- Kivy / KivyMD
- PostgreSQL (psycopg2)
